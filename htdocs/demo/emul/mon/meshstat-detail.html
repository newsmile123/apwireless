This screen displays the statistics related to selected Mesh link.

The following information is displayed below:
MAC Address : MAC address of associated Mesh Unit.
IP Address : IP Address of associated Mesh Unit.
Interface : Interface on which the Mesh Unit is associated.
Protocol : 802.11a, 802.11b, or 802.11g
SNR : Signal-to-Noise Ratio.
Rx Data Rate : Receive data rate.
Tx Data Rate : Transmit data rate.
Rx Errors : Receive errors.
Rx CRC Errors : Receive CRC errors.
Rx Phy Errors : Receive PHY errors.
Tx Errors : Transmit errors.
Tx Excess Retries : Transmit excessive retries.

An access point will no longer be displayed in the list if the link is inactive or has been de-authenticated.
Mesh links are shown in the table as long as the link exist in the AP.
