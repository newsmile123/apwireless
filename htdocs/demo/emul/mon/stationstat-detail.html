This screen displays the statistics related to associated stations and WDS links.

The following information is displayed:
MAC Address : MAC address of associated station or partner MAC address of WDS/Mesh link.
IP Address : IP Address of associated station/Mesh Unit or 0.0.0.0 for WDS links.
Interface : Interface on which the station is associated or the WDS link is configured.
Type : STA (Station) or WDS or MESH
Protocol : 802.11a, 802.11b or 802.11g
SNR : Signal to Noise Ratio.
Rx Data Rate : Receive data rate.
Tx Data Rate : Transmit data rate.
Rx Errors : Receive errors.
Rx CRC Errors : Receive CRC errors.
Rx Phy Errors : Receive PHY errors.
Tx Errors : Transmit errors.
Tx Excess Retries : Transmit excessive retries.
Power Save Mode : Power Save Mode.
Encryption Type : Encryption type used by station.
Authentication Type : Authentication type used by station.
VLAN Id : Station Vlan Id.
QoS : Station using Qos.
TSLF : Time since last frame was received from the associated station or WDS link partner or Mesh.

A station will no longer be displayed in the list, if the client is inactive or has been de-authenticated
WDS links are shown in the table as long as the link is configured in the AP.
