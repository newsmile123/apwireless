Step 6:
Summary

This page is used to configure the management passwords for the access point. It is recommended that you change the SNMP, Telnet (CLI), and HTTP (web) passwords from the default value of "public". These passwords should be changed to a value known only to you. If passwords are not changed, then others may be able to manage the access point and modify its configuration without your knowledge. Valid passwords are between 6-32 characters.

Note: Clicking the Save & Next button will save any changes you made on this screen and take you to the next Setup Wizard configuration page.
