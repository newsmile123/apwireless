Secure Management option allows the use of encrypted and authenticated communication protocols such as SNMPv3, Secure Socket Link (SSL), and Secure Shell (SSH) to manage the Access Point. When Secure Management is turned on, the scope and access for the traditional non-secure means to manage the Access Point is automatically curtailed.

Warning: Make sure to set the SNMPv3 passwords to known values before enabling the Secure Management feature. Once the secure management feature is turned on, the parameters in this page can only be changed or turned off via the serial interface or one of the secure protocols such as SMNPv3, SSL, or SSH.
