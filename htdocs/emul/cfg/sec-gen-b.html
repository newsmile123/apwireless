SSID and Security Data Configuration - Wireless B

Security Profiles are used to configure the allowed security modes. If RADIUS MAC, 802.1x or RADIUS accounting is enabled in the SSID's security profile then the respective RADIUS server profiles should be configured and assigned to this SSID.

Note: Changes to these parameters require access point reboot in order to take effect.
