The spanning tree protocol is an advanced bridge setup option for complex network topologies by eliminating unnecessary data loops. This tab can be used to configure spanning tree protocol characteristics, and bridge priority and path cost.

Warning: Changing these parameters may significantly affect the network topology and performance.

Note: Changes to Hello Time and Forwarding Delay must be in 100 millisecond increments. Changes to Spanning Tree Protocol parameters require access point reboot in order to take effect.
